ClipboardListener

interface ClipboardListener

Listener interface for receiving clipboard change notifications.

Implementations of this interface will be notified whenever the system clipboard content changes. The listener is called on a background thread (platform-dependent), so implementations should handle thread-safety appropriately.

Threading Notes:

  • Android: Called on the main thread (UI thread)

  • JVM (Windows): Called on a dedicated message loop thread

  • JVM (macOS/Linux): Called on a scheduled executor thread

  • JS/WasmJS: Called on the event loop (main thread equivalent)

Important: Exceptions thrown in onClipboardChange are caught internally and will not stop the clipboard monitoring.
