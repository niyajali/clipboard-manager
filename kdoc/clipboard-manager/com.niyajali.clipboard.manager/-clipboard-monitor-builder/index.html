ClipboardMonitorBuilder

class ClipboardMonitorBuilder

Creates configured clipboard monitors with customizable settings.

Provides a convenient way to configure and create clipboard monitors. Use method chaining to set desired options, then call build to create the monitor.

Example:

val monitor = ClipboardMonitor.Builder()
    .setListener(myListener)
    .setDebounceDelay(100)
    .build()
